# Org

## `sfp server org`

Manage Salesforce orgs in sfp server

### Description

The org commands provide functionality for registering, managing, and monitoring Salesforce orgs connected to the SFP server.

### Available Commands

* Register new Salesforce orgs
* List registered orgs
* Update org configuration
* Remove org registrations
* Check org health and status
* Manage org authentication

### Common Use Cases

- Register production orgs
- Manage sandbox environments
- Track scratch org pools
- Monitor org health
- Rotate org credentials
